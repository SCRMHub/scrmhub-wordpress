Sharing Help

Options

Sharing Position

The plugin provides a couple of options for installing the sharing buttons on your site:

  • Automatically
    This is where we will automatically append the buttons into the post output.
    • Top : Above the post content
    • Bottom : Below the post content
    • Both : Above and Below of the post content
  • Manual
    You want to add our code manually to your templates. This provides a much greater level of control and some additional options such as rendering only one button, all buttons, adding settings, etc.

Sharing post types

Choose the types of Wordpress post type the buttons will appear on. By default we pre-tick 'Post' and 'Page', but any custom types will appear. However we cannot guarantee how our buttons will render on those pages as it is depen dant on

Show icon only

By default this is set to 'No', which means it will display a Button with the network's icon and name. Set it to 'Yes' and it will show a simple square icon with the network's logo in it.

Networks

You can enable each network you wish to use here. The noteable difference is the Twitter setting that allows you to specify:

  • Via
    Any shares will have 'via @[value]' appended to the share text.
  • Use Quotes
    This will wrap the text of the tweets in quote marks to make it clear it is the title of an article. This is optional, but something we like doing.

Code Use

Now the good bit... How to use the code in your template. They all render in PHP using Wordpress Hooks but can also be called using RAW PHP.

Render all share buttons

This is the same code the automatic options call, and will render everything as you've defined it in these options.

<?php
	do_action( 'scrmhub_share' );
?>

Render a single share button

If you wish to render a single button, and (optionally) give it a custom label, use the below example.

<?php
	do_action('scrmhub_share_button', array(
		'network' => 'twitter',
		'label' => 'Share to Twitter'
	));
?>

Note: That if you want to render a Twitter button as per this example, but have Twitter disabled on the option page, nothing will render as the system thinks the button is disabled.


Output the URL for sharing to a network

If you want to just output the URL for sharing into your own code for a custom pop-up script for example, this will output it

<?php
	do_action( 'scrmhub_share_url', 'twitter');
?>

Alternatively, if you wish to get the url into your PHP code, you can call it using our global helper function.

<?php
	global $scrmhub_share_url;
    $url = $scrmhub_share_url('facebook');
?>
