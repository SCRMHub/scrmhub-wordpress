Connect Help

Options

Redirect on success

Choose which URL the page goes back to after a login request. This can be overridden on the button using the redirect function. Also note that if a parent_function is specified on the button, this will not get used.

Add to login

Add SCRM Hub buttons to the login form automatically (no programming required). This will show as buttons on the standard login form and will make the parent page refresh to the original re-direct location it would've gone to.

Require Connect to comment

Require users to use the SCRM Hub connect to make comments. This is assuming you are using the built in Wordpress commenting system and not a 3rd party tool.

Set User Photo

Set the user's profile photo to be the one on their social profile when they have successfully connected.

Show the icon only

By default this is set to 'No', which means it will display a Button with the network's icon and name. Set it to 'Yes' and it will show a simple square icon with the network's logo in it.

Code Use

And now, how to use the code in your template. They all render in PHP using Wordpress Hooks but can also be called using RAW PHP.

Render all buttons

Automatically render the login buttons or logout buttons that you've turned on in the config

<?php
do_action( 'scrmhub_connect' );
do_action( 'scrmhub_connect', array(
	'parent_function' => 'scrmhubLoginSuccessTest')
);
do_action( 'scrmhub_connect', array(
	'redirect' => 'https://wherever.com/'
));
?>

Render a single Connect button

<?php
do_action( 'scrmhub_login_button', array(
		'network' => 'facebook',
		'label' => 'Connect with Facebook'
));
?>

Force ender all enabled connect networks

<?php
do_action( 'scrmhub_connect_login' );
?>

Render the logout button

<?php
do_action( 'scrmhub_connect_logout' );
?>

Get the connect URL for a network

If you wish to get the url into your PHP code, you can call it using our global helper function.

<?php
global $scrmhub_connect_url;
echo $scrmhub_connect_url('facebook');
echo $scrmhub_connect_url(array(
	'parent_function' => 'scrmhubLoginSuccessTest',
	'network' => 'facebook'
));
?>

Arguments for connect functions

parent_function
Call a function in the parent window when the connect completes. Will call it with (success, message) where success is boolean and message is string. Note: On mobiles we won't call this as we use redirects. Instead either the redirect override or Redirect on success value will be used.
redirect
This will override the Redirect on success value after completion. Note: If parent_function is specified, that will get called instead.
class
Applied to the scrmhub hub action wrapper
button_class
Applied to the button <a> tag
network
The Social Network to use (currently Facebook, LinkedIn and Twitter)
title
The title for the action box (set to false for not title)
button_label
Applied to the button (we will check the translation files)
